# Routes for Backend

## Authentication
- /userauth/signup
- /userauth/signin
- /userauth/logout

## Products
- /products/create
- /products/get
- /products/delete/:id
- /products/update/:id

## Category
- /category/create
- /category/get
- /category/delete/:id
- /category/update/:id

## Forgot Password
- /forgot/forgot_password
- /forgot/reset_password

## Profile Update
- /updateprofile/profile_update/:id
